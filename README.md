# Usage

This page is the lean webapp version of [Build-a-Batt](https://build-a-batt-webapp.onrender.com/), a project fully parameterizing an electrochemical flow cell. Here, you can find the source code for the webapp which allows adjusting of cell parameters and generation of all flow cell fabrication files.

For the full generation-simulation suite, please take a look at the Build-a-Batt python repository which offers the ability to generate design files, perform hydraulic and electrochemical simulations, and iterate the parameterized design, on your local hardware.
